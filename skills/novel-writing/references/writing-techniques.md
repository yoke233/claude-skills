## 伏笔系统

> 完整手册：`references/伏笔类型与手法.md`

### 六种埋设手法

| 手法 | 说明 | 隐蔽性 |
|------|------|--------|
| 物品伏笔 | 出现物品当时无关紧要，后来成为关键 | 高 |
| 对话伏笔 | 角色不经意的一句话，后来印证或反转 | 高 |
| 行为伏笔 | 角色小动作/习惯暗示隐藏身份 | 高 |
| 环境伏笔 | 场景描写中藏入暗示 | 中 |
| 人物描写伏笔 | 外貌/气质描写暗示真实身份 | 中 |
| 直接预言/谶语 | 预言、梦境、谶语 | 低 |

### 三拍节奏

**埋设（Plant）**→ 双重功能：伏笔在当前场景必须有存在理由，不是硬塞
**强化（Reinforce）**→ 间接提及保持活性，不能埋一次消失 50 章
**回收（Payoff）**→ 最好的回收改变读者对之前所有情节的理解

## 冲突设计

> 完整手册：`references/冲突类型手册.md`

### 七种冲突类型

| 类型 | 说明 |
|------|------|
| 人 vs 人 | 直接对抗（正邪、情敌、门派） |
| 人 vs 自我 | 内心挣扎、道德困境 |
| 人 vs 社会 | 个体与体制/规则的冲突 |
| 人 vs 自然 | 天灾、荒野求生、秘境 |
| 人 vs 超自然 | 天道惩罚、诅咒、命运 |
| 人 vs 科技 | 炼器失控、阵法反噬 |
| 人 vs 命运 | 与预言/宿命的抗争 |

### 冲突升级阶梯

```
表层冲突（个人恩怨）→ 中层冲突（势力对抗）→ 深层冲突（体制碰撞）→ 核心冲突（价值观对决）
```

每层解决同时触发下一层，形成因果链。

## 悬念与钩子

> 完整手册：`references/悬念技法手册.md`

### 五种悬念机制

| 类型 | 机制 |
|------|------|
| 好奇型 | 读者和角色都不知道 |
| 期待型（戏剧反讽） | 读者知道，角色不知道 |
| 紧张型 | 都知道危险，看如何脱身 |
| 意外型 | 制造预期后打破预期 |
| 反预期设计 | 避开最显而易见的走向，用反套路制造意外 |

### 七种章末钩子

| 钩子 | 说明 | 频率 |
|------|------|------|
| 悬崖式 | 最紧张时断章 | 高 |
| 新信息式 | 章末揭示新秘密 | 高 |
| 决定式 | 面临选择不给答案 | 中 |
| 威胁式 | 新危险/敌人出现 | 中 |
| 反转式 | 真相与预期相反 | 低（冲击力大但不能滥用） |
| 期待式 | 预告下章精彩 | 低 |
| 情感式 | 角色关系微妙变化 | 适合情感线 |

**分布规则**：不允许连续 3 章以上同类型。

## 节奏控制

> 完整手册：`references/节奏控制手册.md` + `references/Scene-Sequel模型.md`

### Scene-Sequel 模型（Dwight Swain）

**Scene（动作场景）**：目标 → 冲突 → 灾难（结果比预期更糟）
**Sequel（反应场景）**：反应 → 抉择 → 决定（触发下一个 Scene）

Sequel 长度决定整体节奏：快节奏削减 Sequel，慢节奏拉长。

### 网文节奏模式

**一紧一松，逐步推高**：
```
小冲突 → 小缓冲 → 中冲突 → 中缓冲 → 大冲突（高潮）→ 大缓冲
```

**爽点公式**：铺垫压抑（3-5章）→ 情绪底谷 → 催化事件 → 爽点爆发 → 余韵

**反转渐进节拍**：反转不是开关（0→100），至少需要 4 个节拍：前兆→触发（不完全）→对抗（对手调整）→确立（+代价）。缺少任一节拍会导致断层。

**钩子节奏**：1 章强钩子 + 2 章缓钩子交替。

### 信息控制类

| 装置 | 说明 |
|------|------|
| 戏剧反讽 | 读者知道角色不知道的信息 |
| 不可靠叙述者 | 叙述者视角有偏差或欺骗 |
| 红鲱鱼 | 故意制造的误导线索 |
| 信息限制 | 刻意不告诉读者某些信息 |

### 结构装置类

| 装置 | 说明 |
|------|------|
| 契诃夫之枪 | 前文出现的元素后文必须发挥作用 |
| 麦高芬 | 角色争夺但具体是什么不重要的物品 |
| 首尾呼应 | 开头和结尾形成镜像/对比 |
| 平行叙事 | 两条以上故事线同时推进 |

### 转折装置类

| 装置 | 说明 |
|------|------|
| 突转 | 命运突然逆转，必须由因果链驱动 |
| 发现 | 角色突然认清真相/身份 |
| 天降解围 | 外力解决困境（慎用） |
| In Medias Res | 从故事中间开始讲 |

## 情绪设计

> 完整手册：`references/爽点虐点设计手册.md`

### 四种情绪点

| 类型 | 机制 | 铺垫方式 | 误区 |
|------|------|----------|------|
| 爽点 | 压抑→释放 | 先受委屈 3-5 章再反击 | 无铺垫的碾压不爽 |

## MRU 与微观写作技法

> 完整手册：`references/MRU参考.md`

**动机-反应单元（Dwight Swain）**：

```
动机 = 外部刺激（角色之外发生的事）
反应 = 严格按顺序：感受（本能）→ 动作（身体）→ 对话（理性）
```

对话不能先于感受，动作不能先于本能反应。

### 展示代替叙述

| 问题 | 纠正 |
|------|------|
| 概述句代替场景展示 | 「他感到愤怒」→ 写具体行为/生理反应 |
| 旁白式情感解读 | 让角色行为或他人视角暗示 |
| 能力/设定旁白解说 | 通过角色体验、对话间接传达 |

### 对话技巧

| 要点 | 说明 |
|------|------|
| 标签多样化 | 低声说/吼道/嗤了一声，非千篇一律"说道" |
| 动作拍代替标签 | 2-3 句对话后才加标签，中间用动作拍区分说话人 |
| 角色辨识度 | 遮住名字能分辨说话人，每人独有语言习惯 |

### 感官描写

每场景至少覆盖 3 种感官，视觉之外至少 2 种（嗅觉/触觉/听觉/味觉/本体感觉）。

### 写作修正技法（附录）

| 技法 | 要点 |
|------|------|
| 三层叠加 | 场景层 + 情绪层 + 细节锚点 |
| 做减法 | 删掉读者能自己推断的内容 |
| 留白与不完美 | 段落可断在中间，对话可不给答案 |

| 痕迹 | 规则 |

| 痕迹 | 纠正 |
|------|------|
| 逻辑漏洞与设定矛盾 | 写前加载角色动态状态和章节摘要总表 |

| 痕迹 | 规则 |
|------|------|
| 句长均一 | 刻意交替长短句 |
| 句式起始重复 | 连续 3 句不得同主语/同句式开头 |
| 过渡词过密 | 每段 ≤ 1 个过渡词 |
