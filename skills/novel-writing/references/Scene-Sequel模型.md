# Scene-Sequel 模型

来源：Dwight Swain《Techniques of the Selling Writer》

## 核心概念

所有叙事都由两种基本单元交替组成：**Scene（动作场景）** 和 **Sequel（反应场景）**。

---

## Scene（动作场景）

**结构**：目标 → 冲突 → 灾难

### 目标（Goal）

角色在这个场景中想要达成什么。

**要求**：
- 必须具体、明确（不是「变得更强」，而是「通过这次比试」）
- 读者在场景开始时就知道角色要做什么
- 目标要有时间限制或明确终点

### 冲突（Conflict）

角色追求目标过程中遇到的阻碍。

**要求**：
- 阻碍要不断升级（不是一个障碍反复出现）
- 角色的每次应对都要有代价
- 冲突不能太容易被解决

### 灾难（Disaster）

场景的结果——比角色预期更糟。

**四种灾难等级**：
1. **完全失败**：目标没达成
2. **得不偿失**：达成了但代价太大
3. **新的更大问题**：原问题解决，出现更严重的问题
4. **两难困境**：必须在两个坏选项中选一个

**⚠️**：不是每个 Scene 都必须以灾难结尾——但连续成功的 Scene 会让节奏松弛。

---

## Sequel（反应场景）

**结构**：反应 → 抉择 → 决定

### 反应（Reaction）

角色对灾难的情感和身体反应。

**顺序**（严格遵守）：
1. 情感冲击（「他怔住了」）
2. 身体反应（「双腿一软」）
3. 理性思考（「他意识到……」）

### 抉择（Dilemma）

角色思考接下来怎么办。

**要求**：
- 列出可能的选项（读者能看到）
- 每个选项都有缺陷
- 角色的思考过程要符合其性格

### 决定（Decision）

角色选择了一条路——这成为下一个 Scene 的目标。

---

## Scene 和 Sequel 的节奏关系

**Sequel 长度决定整体节奏**：

| 节奏 | Sequel 长度 | 效果 |
|------|-------------|------|
| 快节奏 | 几句话或省略 | 动作片感，适合战斗/追逐 |
| 正常节奏 | 几段到半章 | 平衡感，适合大部分情节 |
| 慢节奏 | 整章或更长 | 深度感，适合重大转折后 |

**原则**：
- 灾难越大 → Sequel 越长（角色需要时间消化）
- 连续几个短 Sequel 后 → 安排一个长 Sequel（让读者也喘口气）
- 长 Sequel 不能只有思考 → 穿插新信息或日常互动

## 串联示例

```
Scene 1: 主角去集市买灵药（目标）→ 被人抢先（冲突）→ 发现灵药是假的（灾难）
  Sequel 1: 震惊（反应）→ 追查还是放弃（抉择）→ 决定追查（决定）
Scene 2: 主角追查假药来源（目标）→ 被暗中警告（冲突）→ 发现牵涉到门派长老（灾难）
  Sequel 2: 恐惧（反应）→ 揭发还是隐忍（抉择）→ 决定暗中收集证据（决定）
Scene 3: ...
```

## 常见错误

| 错误 | 纠正 |
|------|------|
| Scene 没有明确目标 | 开场就让读者知道角色要做什么 |
| 冲突不够 | 角色太顺利 = 没有 Scene 只有流水账 |
| 灾难后直接进下一个 Scene | 跳过 Sequel = 角色像机器人，没有情感 |
| Sequel 太长变成自言自语 | 穿插外部事件或对话打破内心独白 |
| 所有 Scene 都以「完全失败」结尾 | 适当给予部分成功，让节奏有变化 |
